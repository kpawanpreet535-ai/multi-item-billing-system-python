# Multi-Item Billing System in Python

## Description
A Python-based billing system that allows users to enter multiple items, calculate the total bill, apply discounts, and generate an item-wise invoice using lists and dictionaries.

## Features
- Accept multiple items from the user
- Calculate item-wise amount
- Compute total bill
- Apply discount based on total amount
- Display item-wise invoice
- Show total number of items purchased

## Concepts Used
- Input/Output
- For Loops
- If-Else Statements
- Lists
- Dictionaries

## Sample Output

```
-------FINAL BILL------
Total Bill = 1200
Discount = 120.0
Net Amt = 1080.0

-----Data-----

Name    Price   Qty   Amount
Pen     10      2     20
Book    50      3     150

Total Items = 2
```
